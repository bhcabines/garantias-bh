📤 Como liberar acesso para outros computadores

Após cadastrar ou editar usuários, clique em "Exportar usuarios.js" para baixar o arquivo. Envie esse arquivo para o GitHub (substituindo o usuarios.js existente). Todos os computadores que abrirem o sistema carregarão automaticamente os usuários atualizados.
